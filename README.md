Beautiful animated flower page with cute pastel background.

## Live Demo
Will be on Vercel after deploy.

## How to Deploy to Vercel (Super Easy)

### Option 1: Via GitHub (Recommended)
1. Push this repo to your GitHub (or use this one)
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repo
5. Vercel will auto-deploy (it's just a static HTML)

### Option 2: Quick Deploy (No Git)
1. Go to [vercel.com](https://vercel.com)
2. Drag and drop the `serena.html` file (or the whole folder)
3. Done in 10 seconds
